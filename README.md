# The Badass Riding Academy - Rider IQ Layout Debug Upgrade

This version cleans up the app layout, fixes functional issues, and improves the training flow for easier use and future sales.

## What changed

- Rebuilt the CSS into one cleaner, non-duplicated layout system.
- Improved homepage hierarchy with clearer hero section, stronger CTAs, and a cleaner 3-step path.
- Reworked the assessment screen for better spacing, clearer progress, disabled Continue state, and better selected-answer feedback.
- Fixed progress indicators so answered questions and the current question are visually distinct.
- Fixed the results page so it no longer shows a fake 78% score before the user takes the diagnostic.
- Improved scoring so partial diagnostic results are based only on answered questions.
- Improved weak-spot ranking so unanswered categories do not unfairly appear as failures.
- Improved responsive behavior for tablet and mobile layouts.
- Preserved the Free Lesson expandable-card system and saved Practice Tracker.

## How to run

Open `index.html` in a browser. No build step and no backend are required.

## Launch note

The contact form is still prototype-only. Connect it to email, a CRM, or a form service before using the app as a sales funnel.
